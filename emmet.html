Emmet

Glossaire {
  • 1: Présentation;
  • 2: Syntaxe de base;
  • 3: Placement du curseur;
}

Présentation

Emmet est un plugin des principaux éditeurs de texte (sublime text, brackets, text mate, eclipse, notepad++, Vim...).
Il permet d'accélérer la vitesse de développement en créant de puissantes abbréviations.

Syntaxe de base

body>.main>header>.logo>h1+.img-logo^nav>table>tr>(td>a)*5

Ce qui donne en utilisant la touche tab pour expandre :

<body>
<div class="main">
<header>
<div class="logo">
<h1></h1>
<div class="img-logo"></div>
</div>
<nav>
<table>
<tr>
<td><a href=""></a></td>
<td><a href=""></a></td>
<td><a href=""></a></td>
<td><a href=""></a></td>
<td><a href=""></a></td>
</tr>
</table>
</nav>
</header>
</div>
</body>

Placement du curseur

Utiliser :

${0} ou |

Préférer le deuxième.
