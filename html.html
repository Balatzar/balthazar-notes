Présentation

L'HTML (hyper-text language-markup) est le language de markup du web. Il permet de structurer les pages web.

Il est composé de tags (balises) indiquant un certain type de contenu, qui sera rendu d'une certaine façon par le navigateur.

C'est un language informatique mais pas un language de programmation.

La syntaxe

L'HTML est composé de tags entourant du contenu ou d'autres tags. Ces tags peuvent avoir certaines propriétés.

<tag ouvrant attribut="valeur">Contenu</tag fermant>

Exemple simple

Ce code présente les tags formant un contenant (une div) englobant un titre (h1) et un paragraphe (p).

<div>
<h1></h1>
<p></p>
</div>

Le block

Les tags peuvent être de type block ou inline.

Les tags block occupent automatiquement 100% de l'espace en largeur qu'ils peuvent (en fonction de la taille de leur contenant, que ce soit un autre tag block ou le corps de la page web).

Les block peuvent contenir d'autres block ou des inline.

Exemple : un block avec un tag div (avec une border pour la voir mieux et une hauteur de 20px pour qu'elle ne soit pas une ligne infime)

La div occupe automatiquement 100% en largeur de l'espace que lui accorde son parent (avec un petit espace pour des raisons cosmétiques).

Exemples :

  • div
  • p
  • table
  • header
  • form

Les 4 tags de base

Nom Rôle
!DOCTYPE Le doctype est le premier élément apparaissant dans une page html. Il permet d'indiquer le type de la page html et permet au navigateur d'appliquer les bonnes règles pour la mise en page du document. Par exemple pour l'html 5 (la dernière version) : <!DOCTYPE html>
<html> Indique le début du code html. On lui adjoint en général l'attribut lang pour spécifier la langue du document pour les traducteurs et les logiciels pour les mal-voyants.
<head> Le tag head contient toutes les informations sur la page web : son titre, là où sont stockées les feuilles de style et de script ou des tags <meta> spécifiant par exemple des informations sur la façon de rendre la page.
<body> Le body contient tous les éléments s'affichant dans le navigateur.

Ce qui donne comme base d'un fichier html :

<!DOCTYPE html>
<html>

<head>
Contenu head
</head>

<body>
Contenu body
</body>

</html>

En général on commence à indenter à l'intérieur des tags head et body.

Balises auto-fermantes

Certaines balises sont auto-fermantes, c'est-à-dire qu'elles n'ont pas besoin d'être fermées. Par exemple la balise <!DOCTYPE html> comme plus haut, mais encore la balise <meta> qui permet de spécifier des informations sur la page, ou encore <br> qui est un tag inline permettant de sauter une ligne.

Le flux

Le flux (ou flow) définit la façon dont les éléments vont s'afficher et dans quel ordre. Il est composé de trois règles :

  • Un tag qui s'affiche avant dans le cod s'affichera avant dans le navigateur.
  • Si le texte d’une balise s’allonge, il pousse la suivante plus bas.
  • Une balise imbriquée dans une autre s’affiche par-dessus cette dernière. Exemple : le <li> est par-dessus son parent le <ul>.

Ces règles peuvent être bouleversées par le CSS.

Tag link, script et style

Les tags link et script permettent d'appeller un autre fichier, comme par exemple une feuille de style CSS ou un fichier de script JavaScript.

Le tag script permet aussi d'inclure du code JavaScript directement dans le html, comme le tag style pour le CSS.

CSS :

<link rel="stylesheet" href="url">

<--! ou bien -->

<style>
header {
width: 100%;
}
</style>

JavaScript :

<script src="url"></script>

<--! ou bien -->

<script>
var bonjour = "Coucou !";
alert(bonjour);
</script>

A noter :

  • link est une balise auto-fermante mais pas script.
  • Il est très fortement conseillé d'utiliser des feuilles de style externe plutôt que de mettre le CSS dans le fichier HTML.
