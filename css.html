Centrer une div fixe

Pour centrer une div avec un display: fixe, il faut qu'elle soit à l'intérieur d'une div d'une width: 100% qui aura elle le display: fixe. La première div est alors centrée avec margin: 0 auto.

Démonstration :

HTML

<div class="centreur">
<nav></nav>
</div>

CSS

.centreur {
width: 100%;
position: fixed;
top: 0;
}

nav {
width: 200px;
margin: 0 auto;
}
