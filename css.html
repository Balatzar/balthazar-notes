Centrer une div fixe

Pour centrer une div avec un display: fixe, il faut qu'elle soit à l'intérieur d'une div d'une width: 100% qui aura elle le display: fixe. La première div est alors centrée avec margin: 0 auto.

Démonstration :

HTML

<div class="centreur">
<nav></nav>
</div>

CSS

.centreur {
width: 100%;
position: fixed;
top: 0;
}

nav {
width: 200px;
margin: 0 auto;
}

Z-index

La propriété z-index permet de préciser la position d'éléments sur le plan vertical, c'est-à-dire contrôler leur empilement.

Cette propriété n'agit que sur les éléments positionnés.

Syntaxe :

.element-1 {
z-index: 0;
}

element-2 {
z-index: 1;
}

L'élément 2 est par-dessus l'élément 1.
